# A Continuous-Time Independent Cascade, Activity driven Queue based Discrete-Event Simulation of a Microblogging Social Network

This is the repository of my master thesis. Folders:
1. bskysim: contains the simulation of the CTIC model with Activity driven and queue based timelines.
2. construct-cascades: construts the cascades into an `.ssv` file with the creations, propagations and reposts.
3. validate-trace: a python script that loads _a single run trace_ into memory and checks that any incorrect of prohibited behaviour does not happen. It's a small unit testing to ensure output correctness.
